• Détails sur les récompenses
  • Dès que la préinscription mondiale commencera, vous recevrez de nombreuses récompenses !
  • Les joueurs peuvent réclamer les récompenses via la boîte aux lettres.
  • Les récompenses ne peuvent être remises qu'une fois pour chaque compte d'utilisateur.

Comment participer à l'événement ?

  • Vérifiez le code de parrainage reçu par e-mail après avoir enregistré la préinscription.
  • Partagez le code de parrainage avec vos amis.
  • Laissez-les se préinscrire en utilisant votre code de parrainage.
  • Si 10 amis ou plus se préinscrivent, vous serez invité au salon VIP sur Discord !
  • Diverses récompenses seront offertes dans le salon VIP !
  • Les invitations au salon VIP seront annoncées plus tard via Discord ou l'e-mail avec lequel vous vous êtes préinscrit.
  • Rejoignez-nous sur Discord pour plus d'infos !
